# Agro Forte Futuro Sustentável Equilíbrio entre Produção e Meio Ambiente

Trabalho de escola xd

---
O Agro Forte é uma landing page interativa e chamativa que comunica uma visão de agricultura moderna:
-
alta produtividade e respeito absoluto ao meio ambiente. O objetivo é inspirar produtores, investidores e sociedade a enxergar o agronegócio como motor de regeneração — não destruição — do planeta.

---

### Objetivos do projeto:

Demonstrar que é possível reconciliar escala produtiva com preservação ambiental
Educar sobre práticas como solo vivo, silvopastoril e agricultura de precisão
Engajar visitantes com dados concretos (redução de CO₂, economia de água) e uma calculadora interativa de impacto por hectare
Converter interesse em ação via formulário de contato/newsletter
Recursos e tecnologias usados:

TanStack Start v1 — framework full-stack React com SSR/SSG
React 19 — UI declarativa com hooks modernos
Tailwind CSS v4 — estilização utilitária com design tokens customizados (paleta verde-floresta e tons terrosos)
Framer Motion — animações de entrada, contadores e interações suaves
Lucide React — ícones temáticos (folhas, trator, gotas, etc.)
Imagens geradas por IA — fotos de campos, tecnologia rural e sistemas agroflorestais para dar vida visual ao conteúdo
TanStack Query — gerenciamento de estado server-side
Tipografia: Fraunces (títulos serifados elegantes) + Inter (corpo clean)
A página funciona como um manifesto digital: hero imersivo, pilares interativos, gráfico animado de impacto, calculadora de sustentabilidade e call-to-action final.
